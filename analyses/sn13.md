# Subnet 13 — Data Universe (Macrocosmos)

> Analyzed: 2026-05-04 · easy-entry score at time of analysis: 55.3/100

## TL;DR

Data Universe is the **lowest-friction subnet for a beginner** — no
GPU, ~0.0006 τ registration, mature codebase, run by Macrocosmos (one
of the most respected teams on Bittensor). Mining is **scraping Reddit
and X (Twitter)** into a HuggingFace dataset; reward is based on data
volume + freshness + uniqueness. Earnings are modest (token rewards,
not get-rich), but the on-ramp is the gentlest in the recommended set.

## What it actually does

The subnet builds a public, decentralized social-data dataset. Every
miner runs scrapers that collect:

- **Reddit posts/comments** via PRAW or Reddit's official API
- **X (Twitter) posts** via Apify or X API

Validators sample your local store, verify the data is real (re-fetch
a small subset and compare hashes), and reward you based on:

| Reward dimension | Weight |
|---|---|
| Volume of unique items | high |
| Freshness (recent timestamps) | high |
| Coverage of "valuable" subreddits / accounts | medium |
| De-duplication vs other miners | medium |

Final aggregated dataset is published on HuggingFace by Macrocosmos.

## Reward structure

Top-1 share is moderate (not winner-take-all). The mechanism rewards
*more good data* — there's no lottery, no Arena, no top-N cliff.
Stable, predictable income shape.

## What miners run

- **Hardware:** No GPU. ~4 GB RAM, 50–500 GB disk depending on how
  long you've been mining (storage grows with collected data).
  Bandwidth matters more than CPU.
- **Stack:** Python 3.10+, the `data-universe` repo, `pm2` for process
  management.
- **External services (pick one or more):**
  - **Reddit:** personal Reddit account → API client_id/secret (free)
  - **X:** Apify subscription (~$30–50/month) **or** X API (paid plan
    required after Mar 2023)
- **Time to first reward:** hours, once your store has a few thousand
  items.

## Realistic earnings

At today's emission of ~36 τ/day across 256 miners:

| Setup | TAO/day | USD/day (TAO ≈ $250) |
|---|---|---|
| Reddit-only, default config | ~0.05–0.12 τ | $12–$30 |
| Reddit + X via Apify (mid-pack) | ~0.15–0.30 τ | $35–$75 |
| Tuned subreddit/account list, top-50 | ~0.3–0.6 τ | $75–$150 |
| Top-10 (years of accumulated data) | ~0.6–1.2 τ | $150–$300 |

Subtract ~$30–50/mo if you use Apify for X scraping.

## Honest assessment

**Pros:**

- Cheapest *real* registration on the recommended list (~0.0006 τ,
  basically free relative to other subnets)
- No GPU needed → cheap hardware (a $5/mo VPS works)
- Macrocosmos team has been on Bittensor since the early days; subnet
  is well-documented and stable
- Reward shape is gentle — no Arena, no winner-take-all
- Skill ceiling is configuration-based (pick the right
  subreddits/accounts), not ML-engineering-heavy

**Cons:**

- All slots full → must out-bid an existing UID for entry
- X scraping requires Apify or paid X API → recurring cost eats into
  earnings
- Reddit rate-limits hit fast on a single account; serious miners run
  account pools or accept lower throughput
- GDPR / data compliance policy you must read and follow (real legal
  obligation for the operator)

## Getting started

1. Read the miner guide:
   <https://docs.macrocosmos.ai/subnets/subnet-13-data-universe/subnet-13-mining>
2. Wallet: hot+cold pair, fund with ~0.005 τ buffer (registration is
   essentially free here)
3. Set up Reddit API credentials (free): https://www.reddit.com/prefs/apps
4. Decide on X scraping: Apify (easier, costs money) or skip X
   initially (Reddit-only is fine)
5. Clone the repo, `pip install -e .`, fill in `.env` with API keys,
   edit `scraping_config.json`
6. Run in **offline mode first** for a few hours to build a local store
   before exposing to validators:
   `pm2 start python -- ./neurons/miner.py --offline`
7. Switch to online: `pm2 start python -- ./neurons/miner.py --wallet.name <cold> --wallet.hotkey <hot>`
8. `btcli subnet register --netuid 13 --wallet.name <cold> --wallet.hotkey <hot>`
9. Watch your score climb as your store grows
