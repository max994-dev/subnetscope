# Subnet 33 — ReadyAI (Conversational Data)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 50.9/100

## TL;DR

ReadyAI (by Afterparty) builds structured dialogue datasets for conversational
AI training. Miners generate and curate high-quality conversation data. In
2026 they launched an **agentic transcription system** that autonomously
discovers, retrieves, and transcribes public conversations (audio, forums,
podcasts) at scale — miners now contribute to this pipeline. No GPU required,
very cheap burn fee (~0.0005 τ), 235 active miners is crowded but earnings
are stable.

## What it actually does

Validators issue tasks in two modes:

1. **Generative:** Create a multi-turn conversation on topic X with qualities
   Y, Z (relevance, engagement, novelty, coherence, fluency are scored)
2. **Transcription pipeline (2026):** Process an audio clip or web conversation
   through an agentic transcription workflow — transcribe, tag to a category,
   enrich with structure

Miners submit results; validators score on the 5-quality rubric. The agentic
system routes different task types to miners based on their tracked domain
performance.

## Reward structure

Proportional to quality scores. No winner-take-all. 235 active miners share
~36 τ/day — mid-tier density. Validators weight recent performance more
(rolling window). Miners who specialize in specific topic domains get routed
those tasks preferentially.

## What miners run

- **Hardware:** No GPU for text generation. An LLM API call (GPT-4o, Claude,
  etc.) or a locally-hosted small LLM (Llama 3.1 8B on CPU) works for the
  generative task.
- **For transcription tasks:** A Whisper model running locally (Whisper large-v3
  requires ~10 GB VRAM, but smaller Whisper models run on CPU at slower speed).
- **Stack:** Python, the ReadyAI repo, PM2.

## Realistic earnings

At ~36 τ/day across 235 miners:

| Outcome | TAO/day | USD/day (TAO ≈ $250) |
|---|---|---|
| Generic baseline output | ~0.05–0.10 τ | $12–$25 |
| Good quality + domain focus | ~0.12–0.25 τ | $30–$62 |
| Top-50 (consistent + specialized) | ~0.3–0.5 τ | $75–$125 |

## Honest assessment

**Pros:**
- Near-zero burn fee
- No GPU required (text tasks)
- Stable subnet with an active real-world product (Afterparty is a live
  company with content/audio assets)
- Agentic pipeline is genuinely interesting — more autonomous than manual
  task submission

**Cons:**
- 235 miners = crowded, moderate per-miner emission
- LLM API costs can eat into earnings if you use GPT-4 for generation
- Quality rubric is somewhat subjective — hard to know exactly what scores well
