# Subnet 51 — lium.io (Decentralized GPU Compute)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 54.3/100

## TL;DR

Lium is a **decentralized GPU rental marketplace** — you provide GPU hardware
and get paid in TAO. The registration fee is very cheap (~0.005 τ), but the
catch is that you need actual GPU hardware to earn serious rewards. The 38
active miners (vs 256 slots) means competition is low — only 38 people are
actively contributing. A Sysbox-enabled Docker setup gives a +20% bonus.

## What it actually does

Lium connects GPU renters with GPU providers. Miners run "executor nodes"
(GPU machines) orchestrated by a small central "miner server" (CPU). When
a validator or user requests compute:

1. Validator queries miner about available GPU resources (type, VRAM, bandwidth)
2. Miner provisions a container on the executor node
3. Workload runs, output is returned
4. Validator scores based on: GPU type, GPU count, bandwidth speed, actual throughput

Scoring weights are published in the subnet config — H100 > A100 > RTX 4090 > etc.
The subnet is operated by Datura AI / Celium Compute.

## Reward structure

Validators rank miners by composite score (GPU class × count × bandwidth ×
uptime). Emission is proportional to score. Sysbox runtime enabled on the
executor gives +20% bonus incentive. Only 38 miners are active, meaning the
emission per active miner is generous if you have decent hardware.

## What miners run

- **Miner server (CPU):** 4 cores, 8 GB RAM, 50 GB disk — the orchestration
  layer. One per operator.
- **Executor nodes (GPU machines):** NVIDIA GPU, Ubuntu, Docker + NVIDIA
  container toolkit, 100 GB free disk, 50 Mbps+, 8 GB RAM available.
- **Optional:** Sysbox runtime for +20% bonus.
- **Setup:** Clone `github.com/Datura-ai/compute-subnet`, run install script,
  configure `.env` (wallet, external IP), `docker compose up -d`.

## Realistic earnings

With only 38 active miners sharing subnet emission (~36 τ/day):

| Hardware | TAO/day | USD/day (TAO ≈ $250) |
|---|---|---|
| RTX 3090 / 4090 (1 GPU) | ~0.3–0.7 τ | $75–$175 |
| A100 (1 GPU) | ~0.6–1.2 τ | $150–$300 |
| H100 (1 GPU) | ~1.0–2.0 τ | $250–$500 |
| Multi-GPU node | proportional+ | higher |

These are rough estimates — if more miners join, the per-miner share drops.

## Honest assessment

**Pros:**
- Very cheap registration (~0.005 τ)
- Only 38 active miners → each GPU earns more than on crowded subnets
- No ML modeling required — just provide raw GPU compute
- Open slots in the 256-UID pool means no eviction needed

**Cons:**
- You need physical GPU hardware — this is not a software-only subnet
- GPU hardware costs money upfront and has electricity costs
- "varies" GPU need means any GPU type works but higher-end earns much more
- Datura AI / Celium are less established than Macrocosmos or NATIX

## Getting started

1. Prepare a GPU machine (any modern NVIDIA GPU, Ubuntu, Docker)
2. Install NVIDIA Container Toolkit: `nvidia-ctk runtime configure`
3. Clone: `git clone https://github.com/Datura-ai/compute-subnet`
4. Follow the Celium docs: <https://celiumcompute.ai/docs/bittensor-subnet/miner/overview>
5. Configure `.env`: wallet name, hotkey, external IP, ports
6. Optionally install Sysbox for +20% bonus
7. `docker compose up -d`
8. `btcli subnet register --netuid 51 --wallet.name <cold> --wallet.hotkey <hot>`
