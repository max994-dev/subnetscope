# Subnet 75 — Hippius (Decentralized Cloud Storage)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 46.1/100

## TL;DR

Hippius is a **decentralized cloud storage and compute** subnet — IPFS + S3-
compatible storage, run on a dedicated Hippius Chain that bridges TAO via
hAlpha tokens. Miners earn 41% of subnet emissions based on actual storage GB
and CPU-hour contributions. Burn fee is mid (~0.10 τ), 153 active miners.
The token economics are more complex than most subnets (bridge, lock, burn
mechanics), but the product has real revenue from paying users.

## What it actually does

Hippius provides:
- **IPFS-compatible decentralized storage** — pin files across the network
- **S3-compatible buckets** — standard S3 API so apps can integrate without
  custom client code
- **Compute** — CPU-hour contributions for light workloads

Validators score miners based on:
- Storage GB contributed and actually retrievable (availability test)
- Retrieval speed and uptime
- CPU-hour contributions (for compute tasks)

In 2026 Hippius updated its reward model to **match real network contribution**:
miners are no longer rewarded equally but proportionally to actual GB served
and CPU hours delivered. Unused emissions are burned.

## Token mechanics (important)

Hippius is more complex than typical subnets:
- Mine and earn **Alpha tokens** (standard Bittensor)
- Bridge Alpha 1:1 to **hAlpha** on the Hippius Chain
- **Lock hAlpha** as miner collateral — this reduces circulating supply, lifting
  token price, and is required to operate
- Locked hAlpha means you forfeit dividends, but mining profits should exceed
  that loss

Users pay for storage/compute via credit card or TAO; revenue is distributed
to miners and validators.

## What miners run

- **Hardware:** Disk-heavy. No GPU needed. 100 GB+ SSD (ideally 1 TB+) for
  storage contribution. Decent CPU, 16+ GB RAM. Good upload bandwidth (the
  more you serve, the more you earn).
- **Stack:** Hippius node software (available on hippius.com), Docker.
- **Bridge:** You need to interact with the Hippius Chain bridge to convert
  Alpha → hAlpha and lock it.

## Realistic earnings

At ~36 τ/day to the subnet, miners get 41% (~14.8 τ/day across 153 miners):

| Setup | TAO/day | USD/day (TAO ≈ $250) |
|---|---|---|
| 100 GB disk, modest bandwidth | ~0.05–0.10 τ | $12–$25 |
| 1 TB disk, good bandwidth | ~0.15–0.30 τ | $37–$75 |
| Multi-TB, datacenter bandwidth | ~0.4–0.8 τ | $100–$200 |

Plus revenue from actual paying storage users (credit card / TAO payments).

## Honest assessment

**Pros:**
- Real product with real paying users — not just validator-driven synthetic rewards
- Revenue from user payments supplements TAO emission
- Storage-focused = cheap hardware (disk >> GPU in cost efficiency)
- Referral program for organic growth

**Cons:**
- Bridge + hAlpha lock mechanics add complexity
- 0.10 τ burn fee is mid-tier
- 153 miners is moderately crowded
- Need significant disk and bandwidth to earn meaningfully

## Getting started

1. Website: <https://hippius.com>
2. Docs: <https://hippius.com/alphanomics>
3. Wallet + ~0.12 τ buffer
4. Provision a machine with 500 GB+ SSD and solid uplink bandwidth
5. Install Hippius node software and join the network
6. Bridge Alpha → hAlpha, lock as required collateral
7. `btcli subnet register --netuid 75 --wallet.name <cold> --wallet.hotkey <hot>`
