# Subnet 88 — Investing (88 Quant Fund)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 47.1/100

## TL;DR

Investing88 is a **decentralized portfolio strategy competition** run by
Mobius Fund — miners submit trading strategies for crypto, forex, and US
equities, and the best strategies are aggregated into a real hedge fund (88
Quant Fund) launched Dec 2025. Notably: **no coding required** — miners can
use a zero-code dashboard to submit strategies. Burn fee mid-tier (~0.10 τ),
168 active miners, no GPU needed. The "novelty incentive" rewards differentiated
strategies to prevent everyone copying the top miner.

## What it actually does

Miners act as strategy contributors. Each miner submits portfolio allocations
and trading signals across:

- **TAO/ALPHA staking** — how to allocate bittensor network positions
- **US stocks** — long/short positions on equities
- **Crypto** — BTC, ETH, altcoin positions with leverage

Validators score strategies daily (scoring runs after midnight UTC) on:

| Metric | What it measures |
|---|---|
| Return | Absolute and relative gains |
| Volatility | Stability of returns |
| Drawdown | Maximum peak-to-trough loss |
| Slippage | Cost of executing the strategy at real prices |
| Novelty | How different your strategy is from similar existing ones |

Strategies that closely resemble already-submitted ones get a reduced novelty
score — you're incentivized to think differently, not copy-paste.

The real-world 88 Quant Fund aggregates top strategies and deploys them with
real capital (target: $10K TAO AUM by end of 2026, revenue via 2% AUM fee +
20% profit share).

## Reward structure

Proportional to composite daily score. 168 miners split ~36 τ/day. Rolling
daily scoring means no 10-day lag like Synth — you see results next day.

## What miners run

- **Hardware:** No GPU, no GPU. Pure strategy/data work.
- **Options:**
  1. **Zero-code dashboard** — point-and-click strategy builder on
     investing88.ai (no coding at all)
  2. **API** — submit strategies programmatically via REST endpoint
  3. **Code-based model** — implement a quantitative model using the GitHub repo
- **Data sources:** Any financial data you want to use (TradingView, Yahoo
  Finance, Quant APIs, your own models).

## Honest assessment

**Pros:**
- Lowest technical barrier on the list — zero-code dashboard works
- Daily feedback (unlike 10-day rolling windows on other subnets)
- Real-world capital deployment creates extra revenue path
- Novelty incentive prevents winner-take-all copy-paste dynamics

**Cons:**
- 0.10 τ burn fee is mid-tier
- 168 miners is moderately crowded
- Strategy quality is hard to predict — quant finance is competitive
- Real capital at risk in the fund (you're not just playing with tokens)
