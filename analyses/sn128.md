# Subnet 128 — ByteLeap (GPU Cloud Marketplace)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 39.5/100

## TL;DR

ByteLeap is a **decentralized GPU cloud marketplace** — miners aggregate GPU
workers and rent them out for AI training and inference. Uses bare-metal GPU
pass-through (VFIO) for true hardware performance. Near-zero burn fee
(~0.0005 τ), 41 active miners, 1,216 GPUs deployed network-wide as of Jan
2026. A real product with real compute workloads, but GPU hardware is required
and lease revenue dominates — idle hardware earns almost nothing.

## What it actually does

Miners run two components:

1. **Aggregator (miner server):** CPU-based orchestration layer that advertises
   available GPU resources and manages the rental workflow
2. **Workers (GPU machines):** Actual GPU hardware that executes leased
   workloads via Docker containers with GPU pass-through (VFIO)

Validators score miners via two mechanisms:

| Mechanism | When | What it measures |
|---|---|---|
| Lease score (primary) | Workers rented | Actual compute delivered per lease period |
| Challenge score (secondary) | Idle workers | CPU/GPU matrix multiplication benchmark |

An **Availability Multiplier** based on 169-hour uptime window scales both
scores — consistent uptime matters.

Supported hardware: RTX 3090, 4090, 5090, A100, H100, H200, B200.

## Reward structure

Weighted combination of lease + challenge scores × availability multiplier.
Idle workers earn significantly less than rented workers. 41 miners as of
analysis — relatively low competition for a GPU marketplace. Miners who
attract real leases earn much more.

## What miners run

- **Hardware:** NVIDIA GPU (consumer or datacenter). VFIO GPU pass-through
  recommended for full performance (requires motherboard/BIOS support).
- **Software:** Docker + NVIDIA Container Toolkit, the ByteLeap miner
  (`byteleapai/byteleap-Miner`). Must expose stable external IP.
- **Uptime:** 169-hour availability window — you need 24/7 reliable operation.

## Realistic earnings

Per 1× GPU (estimate, depends heavily on lease demand):

| Hardware | TAO/day (leased) | TAO/day (idle) | USD/day (leased) |
|---|---|---|---|
| RTX 4090 | ~0.5–1.0 τ | ~0.1 τ | $125–$250 |
| A100 80GB | ~1.0–2.0 τ | ~0.2 τ | $250–$500 |
| H100 | ~2.0–4.0 τ | ~0.3 τ | $500–$1000 |

## Honest assessment

**Pros:**
- Near-zero burn fee
- Real workloads from real users (not just synthetic benchmarks)
- Bare-metal performance via VFIO
- 1,216 GPUs already deployed — ecosystem exists

**Cons:**
- Requires physical GPU hardware
- Idle hardware earns very little — you need lease demand
- 24/7 uptime needed for full availability multiplier
- VFIO setup is technically involved (BIOS, IOMMU groups, kernel config)

## Getting started

1. GitHub (miner): <https://github.com/byteleapai/byteleap-Miner>
2. Medium intro: <https://medium.com/@byteleap.ai/byteleap-revolutionary-practice-in-building-the-decentralized-gpu-network-325cbf542120>
3. Wallet + ~0.005 τ buffer
4. Enable VFIO/IOMMU in BIOS (for pass-through) or run Docker without VFIO
   for simpler setup with slightly lower performance
5. Install NVIDIA Container Toolkit, clone the miner repo, configure `.env`
6. `btcli subnet register --netuid 128 --wallet.name <cold> --wallet.hotkey <hot>`
