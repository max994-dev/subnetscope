# Subnet 68 — NOVA (Drug Discovery)

> Analyzed: 2026-05-05 · easy-entry score at time of analysis: 40.9/100

## TL;DR

NOVA is a **drug discovery subnet** — miners run ML-based virtual screening to
find therapeutic molecular compounds. In 2026 NOVA moved to a **bounty-based
reward model** where rewards only pay out when a miner finds a meaningfully
better compound than the current best. Only 2 active miners out of 256 slots,
192 free. Medium GPU required. The near-zero activity could mean massive
opportunity or an effectively dormant subnet — verify before committing.

## What it actually does

Miners use the **PSICHIC** ML model (a graph neural network designed for
protein-ligand binding affinity prediction) to screen molecules from the
Metanova/SAVI-2020 molecular database against protein targets.

Each miner:
1. Receives a target protein from the validator
2. Screens molecules from the SAVI-2020 database
3. Returns top candidates scored by: `target_affinity − antitarget_weighted_affinity + entropy_bonus − repeat_penalty`

Validators verify results and award:
- **Bounty (2026 model):** rewards only release when a miner finds a compound
  that meaningfully improves on the current best known binder
- **Winner weight = 1.0**, all others = 0.0 on-chain — pure winner-take-all

## Reward structure

**Winner-take-all bounty model** (changed from continuous emission in 2026).
With only 2 active miners, whoever finds the better compound wins everything.
But this also means: if no one finds a better compound, no rewards are paid.
The total subnet emission is held in reserve until a threshold-exceeding
discovery is made.

## What miners run

- **Hardware:** CUDA 12.6 GPU (sufficient VRAM for PSICHIC), Ubuntu 24.04,
  Python 3.12. A mid-tier GPU (RTX 3090 / A10) is enough for screening.
- **Stack:** Python, the `metanova-labs/nova` repo, PSICHIC model.
- **Compute:** Molecular screening is parallelizable — more GPU parallelism =
  screen more molecules per hour.

## Realistic earnings

Bounty model makes "daily earnings" not the right frame. Better to think:
"what's the prize if I win a bounty?"

| Scenario | TAO reward | USD equivalent |
|---|---|---|
| Win a bounty (1 of 2 miners) | Very high (accumulated emission) | $thousands+ |
| Don't improve on current best | $0 | $0 |

## Honest assessment

**Pros:**
- Only 2 active miners and 192 free slots — essentially no competition right now
- Near-zero burn fee
- Genuinely exciting: drug discovery is one of the most impactful applications
  of AI
- Large open slot count means easy registration

**Cons:**
- 2026 bounty model = $0 most days; income is lumpy and uncertain
- winner-take-all: if a stronger miner shows up, you earn nothing
- Very small active community — risk of subnet abandonment
- Medium GPU required; molecular screening needs CUDA

## Getting started

4. Install PSICHIC and run a local test screen against the SAVI-2020 database
5. Verify there is active validator activity before registering (check
   on-chain metagraph for recent weight updates)
6. `btcli subnet register --netuid 68 --wallet.name <cold> --wallet.hotkey <hot>`
